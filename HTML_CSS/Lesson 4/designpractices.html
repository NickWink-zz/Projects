Design Practices

Good Practices

Bad Practices

Background Information
The site has an audience(people that travel)
The subject is clear
Page Layout
Appealing to the target audience
Informative page title & page footer area
Repetitive information & balance of text/graphics
Navigation
Main navigation links are clear, consistent, and easy to use
Navigation is structured
All navigation hyperlinks work
Color and Graphics
Color is used consistently with good contrast
Use of color and graphics enhances the site
Each graphic used serves a clear purpose
Functionality
All internal hyperlinks work
All external hyperlinks work
All forms function as expected
Content Presentation
Content provides links to other useful sites
Hyperlinks use a consistent set of colors to indicate visited/non-visited sites
Timeliness: The date of the last revision and/or copyright date is accurate
Browser Compatibility
Displays on popular/current versions of IE (9+)
Displays on popular versions of Firefox
Displays on popular versions of Safari
Displays on popular mobile devices
Multimedia
There is no other multimedia besides pictures
Download times for audio or video files are indicated
Accessibility
Navigation aids, such as site map, skip navigation link, or breadcrumbs are used
Color is not used alone to convey meaning
If images or Flash is the main navigation, clear text links are in the footer section of the page

Reading through my book has improved my ability to understand how important it is to have tools in place for people with special needs. I have an understanding of what it means to have accessibility options available for all people to have the enjoyment of reading your site.

Some of the changes I would make to my site is to have a site map in the footer, and also give the ability to get back to the top of the page without having to scroll and/or tab up. I want everyone to be able to read the site as if they had no special needs and also help interact if so for more improvements.
